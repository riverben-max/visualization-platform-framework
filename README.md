## 前端运行

```bash
# 克隆项目
git clone http://gitlab.neumooc.com/yiyanghealthy/healthy-management.git

# 进入项目目录
cd healthy-management

# 安装依赖
yarn install

# 启动服务
yarn run dev

# 构建测试环境 yarn build:stage
# 构建生产环境 yarn build:prod
```
